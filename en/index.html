Catalogo De Filme API

Programming Language: C# em ASPNET Core WebAPI

I developed a Movie Catalog API using ASP.NET Core, implementing the Repository pattern and Mediator with CQRS to separate commands and queries. I used DDD (Domain-Driven Design) to structure the project efficiently and AutoMapper to simplify object mapping. Data persistence is handled with Dapper, connecting to a SQL Server database. The API also integrates ILogger for robust log management, ensuring scalability and maintainability. This project exemplifies a modern and solid architecture for web applications.

Minimal-API Cadastro de Veículos

Linguagem: C# em ASPNET Core

Desenvolvi uma Minimal API para cadastro de veículos em ASP.NET Core, utilizando o padrão Repository e DDD (Domain-Driven Design). A aplicação se conecta a um banco de dados MySQL via Docker e emprega Entity Framework para gerenciamento de dados. Implementei autorização e autenticação com JWT Bearer, permitindo controle de acesso para papéis de Administrador e Editor. Este projeto demonstra uma abordagem moderna e segura para construção de APIs.
