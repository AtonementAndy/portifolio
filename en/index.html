Movie Catalog API

Stack: C# em ASPNET Core WebAPI

I developed a Movie Catalog API using ASP.NET Core, implementing the Repository pattern and Mediator with CQRS to separate commands and queries. I used DDD (Domain-Driven Design) to structure the project efficiently and AutoMapper to simplify object mapping. Data persistence is handled with Dapper, connecting to a SQL Server database. The API also integrates ILogger for robust log management, ensuring scalability and maintainability. This project exemplifies a modern and solid architecture for web applications.

Minimal-API Vehicle

Stack: C# em ASPNET Core

Developed a Minimal API for vehicle registration in ASP.NET Core, using standard Repository and DDD (Domain-Driven Design). The application connects to a MySQL database via Docker and employment Entity Framework for data management. I implemented authorization and authentication with JWT Bearer allowing access control for Administrator and Editor roles. This project demonstrates a modern approach and security for building APIs.
