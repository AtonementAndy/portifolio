Catalogo De Filme API

Linguagem: C# em ASPNET Core WebAPI

Desenvolvi uma API de Catálogo de Filmes utilizando ASP.NET Core, que implementa o padrão Repository e o Mediator com CQRS para separar comandos e consultas. Utilizei o DDD (Domain-Driven Design) para estruturar o projeto de forma eficiente, e o AutoMapper para simplificar a mapeação de objetos. A persistência dos dados é realizada com Dapper, conectando-se a um banco de dados SQL Server. A API também integra o ILogger para um gerenciamento robusto de logs, garantindo escalabilidade e manutenibilidade. Este projeto exemplifica uma arquitetura moderna e sólida para aplicações web.

Minimal-API Cadastro de Veículos

Linguagem: C# em ASPNET Core

Desenvolvi uma Minimal API para cadastro de veículos em ASP.NET Core, utilizando o padrão Repository e DDD (Domain-Driven Design). A aplicação se conecta a um banco de dados MySQL via Docker e emprega Entity Framework para gerenciamento de dados. Implementei autorização e autenticação com JWT Bearer, permitindo controle de acesso para papéis de Administrador e Editor. Este projeto demonstra uma abordagem moderna e segura para construção de APIs.
